Objetivos Técnicos

Base do Explorador

  • Ler pastas do sitema
  • Listar arquivos dinamicamente
  • Diferenciar arquivos e pastas
  • Navegação por rotas dinâmicas
  • Template único reutilizável
  • Botão voltar funcional

Tornar funcional como repositório

  • Downloads de arquivos ao clicar
  • Navegação segura dentro da pasta base
  • Breadcrumb (Caminho Clicável)
  • Tratamento de erros (permição, pasta inexistente)
  • Impedir acesso fora da pasta base

Melhorar a usabilidade (UX para celular)

  • Interface responsiva (CSS)
  • Botões grandes e clicáveis
  • Ícones claros (📁 📄 ⬅️)
  • Barra de navegação simples
  • Layout adaptado para tela pequena

Objetivos de segurança

  • Limitar a uma pasta específica (ex:C:/Repositório)
  • Bloquear acesso com ../
  • Adicionar senhas
  • Não expor o disco inteiro

  • Ordenar arquivos

  • Visualização de imagens e arquivos variáveis
